Система заказов Pattern Brew
============================

Обзор
-----
Проект реализует ядро для управления заказами и простой интерфейс Tkinter.
В качестве паттерна оставлен только «Наблюдатель».
Есть альтернативный интерфейс на PyQt6.

Где реализован паттерн
---------------------
- Наблюдатель: core/patterns/observer/observers.py

Структура проекта
-----------------
coffee_order_system/
  core/        Бизнес-логика, модели, наблюдатели, сервисы
  gui/         Tkinter интерфейс
  tests/       Юнит-тесты
  main.py      Точка входа

Запуск
------
1) Откройте терминал в coffee_order_system
2) Выполните:
   python main.py

Запуск PyQt6
------------
python main_pyqt6.py

Тесты
-----
python -m unittest discover -s tests
